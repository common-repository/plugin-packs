== Description ==

One click plugin bundle installs

All you need to do is create a text file with the plugin slugs, one per row.
Rename it to "whatever.wpp" and then go to the Plugin Packs page in the admin.

The bundle can then be installed through a single click.

See http://horia.me/the-wordpress-plugin-packs/ for more info.

== Installation ==

1. Upload the contents of the archive to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
